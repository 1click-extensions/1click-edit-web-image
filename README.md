# <a href="https://1ce.org/1click-edit-web-image">1CE - 1click edit web image</a>

Edit image from any webpage (locally as long as you choose to share it)

Usage Guide:

1. Install the extension (Click on the "Add to Chrome")
2. After you've installed the extension, open the page that contains the image you want to edit, and click the extension icon.
3. Click a picture you want to edit.
4. On the new page that opens, edit the image and then save it to your computer or share it online.
For more instruction visit the website: https://1ce.org

## Installation

The easiet way to install the plugin, is going to [1CE - 1click edit web image website](https://1ce.org/1click-edit-web-image), and then add it to chrome.

## Developing

1.  Fork the repo.
2.  Clone it to your machine.
3.  Drag and drop the the file into Google chrome browser.
    If you get any error mentioning "Apps extensions and user scripts cannot be added from this website", open chrome://extensions/ in a new tab and repeat step 1

## Install on Chrome (For development)

* Download chrome-extension directory
* Go to "chrome://extensions
* Click "load unpacked extension"
* Choose the directory you downloaded it to.
